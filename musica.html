Reproductor de música

Sistema personalizado para la cración de listas de música

Estado:

inconcluso

Especificaciónes:

Lenguaje PHP, conexión a base de datos.

Descipción:

Un sistema para escucha tu música favorita. El sistema tiene dos tipos de usuario: Administrador y Usuario. Cada usuario podra crear su propia cuenta en el sistema, podra crear listas de canciones y agregar las canciones que el sistema le comparte.

Acceso al proyecto en github:

https://github.com/Patricia14/ReproductoMusica
